***Clone the repository***
```zsh
git clone https://git.codecrafters.io/68636aab87d3936a codecrafters-shell-cpp
cd codecrafters-shell-cpp
```
***Push an empty commit***
```zsh
git commit --allow-empty -m 'test'
git push origin master
```
***Navigate to src/main.cpp***
```zsh
code --goto src/main.cpp .
```
